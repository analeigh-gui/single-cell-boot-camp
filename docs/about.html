Introduction

How to use the materials?

This repository contains the materials for the Single Cell Boot Camp.

It has two chapters of precourse preparation and nine chapters of the course.

Important

For those who are new or need a refresher to R, please review Precourse Preparation - R. If you are familiar with the functions listed in that chapter, you can skip it.

For all participants, please review Precourse Preparation - Theory.

We will focus on Chapters 1 to 9 in the course. Welcome to review them before the course if you have time (though the learning outcome is better if you go through the materials before attending the course - from previous students).

Description

The Single Cell Boot Camp: Analysis of scRNA-seq with R is to provide an overview of the analysis of single-cell RNA-seq using R. It is designed for early-stage researchers who want to adopt single-cell analysis in their toolbox. The workshop consists of theory in presentation and application in hands-on practice.

The workshop starts with a fast-paced training session on the common workflow of single-cell RNA-seq analysis, then continues with sessions on the common workflow of scRNA-seq using bioinformatics packages like Seurat.

Prerequisites

Each participant are expected to have:

1. An introductory background in statistics, specifically in the below concepts: Median, log2 fold change, p-value, adjusted p-value, variance, PCA.

2. Basic familiarity with R and tidyr package, specifically in the below functions: %>% (pipe), filter, mutate, pivot_longer, pivot_wider, map, pmap, select, left_join.

If you would like to learn on your own with publicly available resources:
Chapter 1 - 4 from MordernDive.

If you would like to participate in an instructor-led R course:
R for Biologist from Edinburgh Genomics. For details, please contact Edinburgh Genomics.

Expected outcome

By the end of the workshop, participants will be familiar with the following topics: Common workflow of single cell analysis (pre-processing, quality control, filtering, normalization, Clustering, Cell type annotation).
